Steps to Fix:

  1. Run the Windows Update Troubleshooter.
  2. Clear update cache: stop Windows Update service and delete contents of C:\Windows\SoftwareDistribution.
  3. Restart the service and check for updates again.
  4. Use the Windows Update Assistant from Microsoft's website.
  5. Ensure sufficient disk space for updates.
